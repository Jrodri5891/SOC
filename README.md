# SOC / DFIR Investigation Labs

This repository contains hands-on cybersecurity investigation reports focused on Digital Forensics & Incident Response (DFIR), malware analysis, memory forensics, and network traffic analysis. These labs were completed using real-world analyst workflows and industry-standard tools including Wireshark, Volatility, VirusTotal, and Detect It Easy.

## Included Investigations

### FakeGPT — Malicious Chrome Extension Analysis

Analyzed a malicious browser extension designed for credential theft, keylogging, cookie hijacking, and covert data exfiltration. Investigated JavaScript functionality, anti-analysis behavior, AES encryption, Base64 obfuscation, and command-and-control communication.

### Lockdown — Multi-Stage Intrusion Analysis

Reconstructed a complete attack chain involving IIS exploitation, SMB reconnaissance, web shell deployment, reverse shell communication, persistence mechanisms, memory forensics, and malware attribution to Agent Tesla. Findings were mapped to MITRE ATT&CK techniques using Wireshark, Volatility, and VirusTotal.
